# SilenceCutter

Приложение для автоматической обрезки тишины в видео с использованием FFmpeg.

![SilenceCutter Screenshot](screenshot.png)

## Возможности

- **Автоматическое обнаружение тишины** — находит тишину в начале и конце видео
- **Удаление внутренних пауз** — вырезает длинные паузы между словами
- **Склейка после обрезки** — опция для автоматического объединения обработанных видео
- **Drag & Drop** — перетаскивание файлов и папок
- **Настраиваемые параметры**:
  - Порог тишины (dB)
  - Минимальная длительность паузы
  - Точная обрезка (с перекодированием) или быстрая (без)
- **Пакетная обработка** — обработка множества файлов за раз
- **Изменение порядка** — сортировка видео кнопками ⬆️⬇️ или перетаскиванием

## Установка

### Требования

- Python 3.10+
- FFmpeg (должен быть в PATH)
- PyQt6

### Установка зависимостей

```bash
pip install PyQt6
```

### Установка FFmpeg

**Windows:**
1. Скачать с https://ffmpeg.org/download.html
2. Добавить путь к ffmpeg.exe в переменную PATH

**Linux:**
```bash
sudo apt install ffmpeg
```

**macOS:**
```bash
brew install ffmpeg
```

## Запуск

```bash
cd video_trimmer
python main.py
```

## Использование

1. Перетащите видео файлы в окно приложения или используйте кнопки "Добавить файлы" / "Добавить папку"
2. Расположите видео в нужном порядке (кнопки ⬆️/⬇️ или перетаскивание)
3. Настройте параметры обнаружения тишины:
   - **Порог тишины**: -50 dB (консервативно) до -30 dB (агрессивно)
   - **Мин. длительность**: минимальная длина тишины для обнаружения
4. Выберите опции:
   - **Точная обрезка** — перекодирует видео для точных границ (рекомендуется)
   - **Удалять паузы внутри** — вырезает паузы между словами
   - **Склеить после обрезки** — объединит все видео в один файл
5. Нажмите "Обработать все"

## Параметры FFmpeg

| Параметр | Описание |
|----------|----------|
| `-50 dB` | Только полная тишина |
| `-30 dB` | Тихие звуки тоже считаются тишиной |
| `1 сек` | Игнорировать паузы короче 1 секунды |
